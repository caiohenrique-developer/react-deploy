## [ğŸ”](#top) How To Use :grey_question:

To clone and run this application locally, you'll need [Git](https://git-scm.com), [Node.js v10.16][nodejs] or higher + [Yarn v1.13][yarn] or higher installed on your computer.

<details open>
  <summary>Hit me to toggle collapse: ğŸ•¹ï¸<sup>ğŸ¤</sup></summary>

  _From your `command line` follow these steps..._

  ```bash
    # Clone the project
    $ git clone https://github.com/caiohenrique-developer/react-deploy 'GoBarber Web' && cd 'GoBarber Web'

    # Install dependencies
    $ yarn
    # or
    $ npm i

    # Start the project
    $ yarn start

    # Very well, now you can enjoy this project! ;)
  ```
